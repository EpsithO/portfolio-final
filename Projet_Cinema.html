Cinéma Projet BTS n°2

API-REST

Technologie : Php/Symfony/Mysql/Git

Description

Ce projet consiste à développer une application WEB permettant à des utilisateurs de réserver des séances dans un cinéma. Les fonctionnalités attendues devront être exposées par une API REST avec le Framework Symfony. L’API REST devra se connecter à une base de données MYSQL/MariaDB dans laquelle l’ensemble des données devront être stockées.
